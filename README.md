# Live Fake News Detector 📰

This real-world ML project helps users detect whether a news article is REAL or FAKE based on input text or a URL.

## 🔍 Features
- Accepts either news text or article URL
- Uses trained ML model (TF-IDF + Logistic Regression)
- Simple Flask web interface

## 🚀 Setup Instructions

1. Clone the repo or copy the files
2. Install dependencies:
```bash
pip install -r requirements.txt

```
